ESP32 and DHT11 Setup Guide

Step 1: Code

Hardware connections:

  • dht: SDA to esp: D15
  • dht: VCC to esp: 3V3
  • dht: GND to esp: GND
Download Code

Note- change wifi and thingspeak channel details only in the code. Any other changes not required

Step 2: Adding Index Path in Arduino IDE Preferences

File -> Preferences -> Additional Board Manager Paths

Index Path:

paste path in additional board Manager, Search for 'esp 32' and install the board for ESP32.

Step 3: Install Libraries from Library Manager

Required Libraries:

  • ThingSpeak
  • Optional - DHT Sensor library for ESPx

Select ESP32 WROOM-DA* board and respective port for compiling the code

Try to execute, if port is undetected go for step 4.

Step 4: Driver Installation

Download CP210x VCP Windows Drivers

After download, extract the zip file and install the driver, then recheck compiling code.

Note: This code and drivers may change according to the board/hardware you use. Here, we are using ESP32 WROOM-DA* board and DHT11 sensor.
